Case Study: Solo Development of a Mobile Crypto Calculator for Conexus Crypto

Client Overview

Conexus Crypto, a leading cryptocurrency exchange platform, required a specialized web page to facilitate cryptocurrency transactions in Vietnamese Dong (VND). The objective was to create a mobile-optimized calculator that would enable users to convert VND into various cryptocurrencies, display wallet information for transfers, and forward transaction details via Telegram and Google Sheets.

Project Details

  • Platform: WordPress
  • Technologies Used: PHP, JavaScript, CSS
  • Languages: English, Russian, Vietnamese
  • APIs: Yahoo Finance API, Google Sheets API, Telegram API
  • Live URL: Conexus.vn

Project Scope

User Input for Conversion Amount

Users enter the amount they need in VND (e.g., 5,000,000 VND).

Cryptocurrency Selection

Users choose from five available cryptocurrencies. Each option displays the equivalent amount in cryptocurrency and the respective exchange rate fetched from Yahoo Finance. A variable markup is added to the final cryptocurrency total, based on the purchase amount.

Transaction Details

Users are presented with a QR code, wallet number, and the amount in the chosen cryptocurrency for the transfer.

Transaction Confirmation

Users upload a screenshot of the transfer, which is sent to the operator via Telegram and recorded in Google Sheets for further processing. Alternatively, they upload a URL to the completed transaction.

Thank You Page

Users are redirected to a thank-you page with additional information and contact details for future transactions. Transfers need to be manually confirmed for security reasons, and users are informed of the ETA.

Implementation

Step 1: Conversion Input

A complex input form was created using WP Forms 7 and custom HTML/JS/CSS (support 3 languages) to capture the amount in VND. Input checks were placed (forced numbers, no text, reg exp to exclude some characters). Input data was sanitized thoroughly. A minimum-amount info message was added (allowable transactions needed to exceed 10 USD.).

Example: User inputs 5,000,000 VND.

Step 2: Cryptocurrency Selection

JS scripts fetch exchange rates from Yahoo Finance. Display cryptocurrencies table with respective rates.

Example Rates:

  • USDT TRC20: 200 USDT (1 USDT = 2,500,000 VND)
  • Bitcoin: 0.002 BTC (1 BTC = 1,800,000,000 VND)

Ensured exchange rates were updated automatically in real-time, with minimal calls.

Step 3: Display Final Amount Information

Generated equivalent total to be transferred in the chosen cryptocurrency, after markup and conversion. Displayed wallet numbers and QR code for the chosen amount dynamically.

Example: Final total: 200 USDT. Transfer to Wallet QR or number: 123456.

Step 4: Upload and Confirmation

Integrated a file upload feature for users to submit transfer screenshots. Utilized PHP to handle file uploads and store references in a database. Sent transaction details, including partner UTM tags (fetched form the URL), to the operator via Telegram and Google Sheets API.

Multilingual Support

Implemented language support in English, Russian, and Vietnamese. Texts provided by the client were integrated into the platform. Users could switch languages seamlessly.

Challenges

  • Lack of Yahoo Finance API documentation: Yahoo Finance API is not directly supported by Yahoo anymore. As a workaround, a third-party paid solution (RapidAPI) affiliated with Yahoo Finance had to be integrated instead. Said solution had no documentation. Manual search queries had to be made to the API to find relevant endpoints, to obtain valid rates in real-time, and displayed (and used) as needed within the app.
  • Lack of direct Crypto to Vietnamese Dong conversion rates: Yahoo Finance does not support direct conversions from crypto (like BTC, ETH, USDT) to VND. As a workaround, USD was used as an intermediary currency. Essentially, crypto was first converted to USD, and then VND. This added an extra layer of complexity to the calculations.
  • Complex calculations: Complex calculations, up to 8 decimal places to cover 1 “Satoshi” or smallest allowable BTC amount, had to be added — with a markup to make it feasible for the client to support payments. For example, BTC had to have a minimum transaction amount of USD 50 (converted to VND), with a 4% initial markup. Lower-value transactions had to be disallowed. The markup was reduced to 3% for amounts over $200, and 2% for amounts over $400. Said calculations had to be done for all available cryptocurrencies, with unique markups, and displayed to the visitor in local VND.
  • Telegram API picture requirements: The transaction had to be sent as a picture to Telegram, with various details to be added as caption elements, for manual perusal. To make this work, the picture had to be drawn via Canvas API, converted to base 64, sent to WP PHP server via AJAX requests, sanitized with MIDI checks, and finally forwarded to Telegram.
  • Multi-part Forms: The forms for the transaction had to be added with WP Forms 7, so that the client could modify them later personally, without using code. This added an extra layer of complexity. The forms had to be hidden with CSS, and linked “on click” with JS. It became a pseudo-SPA, in the end.

Outcome

The project was successfully launched and met all client expectations. Key achievements include:

  • User-Friendly Interface: The calculator was intuitive and easy to navigate on mobile devices.
  • Real-Time Conversions: Automatic fetching and updating of exchange rates ensured accurate and real-time conversions.
  • Secure Transactions: QR codes and wallet details were securely generated and displayed.
  • Efficient Processing: Transaction details were promptly sent to the operator, enhancing operational efficiency.
  • Multilingual Accessibility: The page was accessible in three languages, broadening its usability.
  • UTM Network: Automatically integrated into existing UTM network.

Client Feedback

Conexus Crypto Exchange appreciated the timely delivery and the robust functionality of the crypto calculator. They particularly valued the seamless integration with Telegram and Google Sheets, which streamlined their transaction confirmation process.

Future Enhancements

The client has requested multiple enhancements, which will be implemented in the future:

  • Adding support for additional cryptocurrencies.
  • Recreating the program for the SA division.
  • Expanding into a full-fledged crypto wallet and currency exchange service.

This case study highlights the successful development and deployment of a cryptocurrency calculator for Conexus Crypto Exchange by a lone developer. Also, it showcases how everyday WordPress and PHP can create specialized financial tools affordably.
